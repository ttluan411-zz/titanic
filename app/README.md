### Installing the packages

**STEP 1.** Ensure you have Node.js installed.
**STEP 2.** From root run: 

npm install


### Running the server

**STEP1.** From the `./api` folder, run:
```
npm run build
npm run start
```

### Running the app
**STEP 1.** From the `./app` folder, run:
```
npm start
```
